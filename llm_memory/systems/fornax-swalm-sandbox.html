摘要

梳理 Agent Ops 平台、Agent 框架和隔离执行环境之间的边界与集成问题。

中文正文

  • Fornax、Swalm 和 Sandbox 分别覆盖 Agent Ops、agent framework 和隔离执行环境，三者共同支撑 Agentic RL/agent 应用。
  • Fornax 更偏平台和运维视角，关注任务、运行、观测、评估、资产和协作。
  • Swalm 更偏 agent/rollout 运行时，连接模型、工具、轨迹、状态和训练/评估任务。
  • Sandbox 提供隔离执行、资源限制、工具调用边界和安全控制，是 agent 运行质量与安全的底座。
  • 集成重点是统一 trace、artifact、session、权限、网络访问和失败分类，而不是只打通 API。
