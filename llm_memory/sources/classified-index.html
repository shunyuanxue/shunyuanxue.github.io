摘要

本页按照 wiki 的主题划分，把原始 .docx、.pdf、.png、.drawio source documents 组织成分层索引。原始文件仍保留在仓库根目录和 lark_docs/，本页只建立阅读路径和维护入口。

使用方式

  • 先按主题找到相关原始文档，再进入关联 wiki 页面看中文结论。
  • 同一 source document 可以出现在多个主题下，因为 PD/KV、DeepSeek、硬件网络、稳定性等经常交叉。
  • 需人工/OCR 复核 和 低抽取 的文档不能直接作为强结论来源，需要回到原图、PDF 或人工读图确认。
  • 本页在私有版本中会展开每篇 source document 的章节入口和代表性原文片段；公开版本会剥离这些 private detail。
  • 完整平铺目录仍在 源文件目录，重复文件和治理状态见 源文件治理。
