摘要

总结 QwenNext/Kimi 等 hybrid linear-attention 模型在 GDN/KDA state cache、chunkwise/recurrent compute 和 prefix cache 上的适配。

中文正文

  • Linear Attention / hybrid attention 把部分层的序列增长 KV 换成 recurrent state，目标是降低长上下文成本。
  • QwenNext、Kimi 等模型会同时存在标准 attention cache、GDN/KDA state、SSM/Conv cache，因此 serving runtime 需要多状态缓存。
  • chunkwise 与 recurrent 计算路径对 scheduler 和 prefix cache 的要求不同，不能直接套用标准 PagedAttention 假设。
  • 框架适配要处理 layer routing、cache index、一致性、prefill/decode 切换和混合层 kernel fallback。
  • 这种模型结构优化的收益要用端到端请求形态验证，尤其是长上下文、多轮对话和 batch 混合场景。
